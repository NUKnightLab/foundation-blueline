Blueline is the style guide for Knight Lab web pages, apps and projects. It is based on Foundation 5. This page itself is built with Blueline and Foundation.

To use Blueline in your project, simply include the below links in your project. See the rest of this guide for proper usage of the included components.

[links]

To develop Blueline, see the README on GitHub.

There are some advanced features available. You will need to include files for these separately.

[links]

Headlines

See the Soundcite project page for an example of how to use each heading.

Rendered HTML

h1. Used on the homepage, no more than two lines long.

h3. Used for calls to action.

Paragraph styles

By default body text, like this, is 16px. Foundation calculates sizes in rems, or root ems, so this base size is what all font sizes are calculated from. Media queries shrink the body text size and, by relation, all other font sizes.

Rendered HTML

By default body text, like this, is 16px.

HTML

  <p>By default body text, like this, is 16px.</p>
Rendered HTML

Lead paragraphs are used as introductions to projects or other large pages.

HTML

  <p class="lead">Lead paragraphs are used as introductions to projects or other large pages.</p>

Layout

Grid system

Foundation's grid system is based on rows and columns. *Blueline's rows have a max-width of 1200px. If you want to allow your content to be wider than 1200px, simply omit the use of the row div and use only the column classes. You do not need to use columns within rows. *However, you may find it helpful to use .row.full-width if you are having difficulty aligning columns.
